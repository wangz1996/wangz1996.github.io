# 图片目录说明

本目录用于存放个人主页所需的图片文件。

## 目录结构

```
images/
├── profile/          # 个人照片
│   └── photo.jpg     # 个人头像照片（推荐尺寸：400x400px，圆形显示）
└── projects/         # 项目相关图片
    ├── atlas.jpg     # ATLAS实验相关图片
    ├── cepc.jpg      # CEPC项目相关图片
    ├── darkshine.jpg # DarkSHINE项目相关图片
    ├── vlast.jpg     # VLAST探路者项目相关图片
    ├── ahcal.jpg     # AHCAL能量重建深度学习系统相关图片
    └── aetheris.jpg  # Aetheris多智能体助手系统相关图片
```

## 图片要求

### 个人照片 (images/profile/photo.jpg 或 photo.png)
- **文件名**: `photo.jpg` 或 `photo.png`（两种格式都支持，系统会自动检测）
- **推荐尺寸**: 400x400 像素或更大（正方形）
- **格式**: JPG 或 PNG
- **说明**: 照片将显示为圆形头像，建议使用清晰、专业的个人照片。系统会先尝试加载jpg格式，如果不存在则自动尝试png格式

### 项目图片 (images/projects/)
- **文件名**: 支持 `.jpg` 或 `.png` 两种格式（系统会自动检测）
- **推荐尺寸**: 1200x675 像素（16:9 比例）或类似比例
- **格式**: JPG 或 PNG
- **说明**: 
  - 图片将自动适应容器大小
  - 建议使用与项目相关的图片，如实验装置、数据分析结果、系统界面等
  - 系统会先尝试加载jpg格式，如果不存在则自动尝试png格式
  - 如果两种格式都不存在，页面会显示占位符提示

## 如何添加图片

1. **个人照片**:
   - 将您的个人照片命名为 `photo.jpg` 或 `photo.png`（两种格式都支持）
   - 放置到 `images/profile/` 目录下
   - 系统会自动检测并使用可用的格式

2. **项目图片**:
   - 将各项目的相关图片按照上述文件名命名（可以使用 `.jpg` 或 `.png` 扩展名）
   - 例如：`vlast.jpg` 或 `vlast.png` 都可以
   - 放置到 `images/projects/` 目录下
   - 系统会自动检测并使用可用的格式
   - 如果某个项目暂时没有图片，可以暂时不添加，页面会显示占位符

## 图片优化建议

- 使用图片压缩工具（如 TinyPNG）减小文件大小，提高页面加载速度
- 确保图片清晰度足够，但文件大小控制在合理范围内（建议每张图片 < 500KB）
- 对于项目图片，建议使用与项目内容相关的专业图片

## 注意事项

- 所有图片路径都是相对路径，确保图片文件放在正确的目录下
- **支持格式**: 所有图片都支持 `.jpg` 和 `.png` 两种格式，系统会自动检测
- 如果图片文件不存在，页面会自动显示占位符，不会出现错误
- 建议使用 JPG 格式以获得更好的压缩比，PNG 格式适合需要透明背景的图片
- 同一张图片只需要提供一种格式即可（jpg或png），不需要同时提供两种格式
